# Featuresets

**Featureset** is a string literal that allows showing or hiding UI elements and changing the chart's behavior.

## How to use[​](#how-to-use "Direct link to How to use")

To enable or disable a feature, include it in the [`enabled_features`](/charting-library-docs/v29/api/interfaces/Charting_Library.ChartingLibraryWidgetOptions#enabled_features) or [`disabled_features`](/charting-library-docs/v29/api/interfaces/Charting_Library.ChartingLibraryWidgetOptions#disabled_features) array of the [Widget Constructor](/charting-library-docs/v29/core_concepts/Widget-Constructor) object.

```
<script>  
    new TradingView.widget({  
        container: 'chartContainer',  
        locale: 'en',  
        library_path: 'charting_library/',  
        datafeed: new Datafeeds.UDFCompatibleDatafeed("https://demo-feed-data.tradingview.com"),  
        symbol: 'AAPL',  
        interval: '1D',  
        enabled_features: ["show_spread_operators"],  
        disabled_features: ["items_favoriting", "show_object_tree"]  
    });  
</script>
```

This code sample enables the spread operators in the [*Symbol Search*](/charting-library-docs/v29/ui_elements/Symbol-Search) dialog
and disables [*Add to favorites*](/charting-library-docs/v29/ui_elements/drawings/#favorite-tools) and *Show Object tree* buttons in the UI.
