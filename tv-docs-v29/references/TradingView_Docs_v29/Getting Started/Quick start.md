# Quick start

## Getting access[​](#getting-access "Direct link to Getting access")

The [Advanced Charts](https://github.com/tradingview/charting_library) and [Trading Platform](https://github.com/tradingview/trading_platform) repositories are private, and you should fill out the form on the [TradingView website](https://www.tradingview.com/HTML5-stock-forex-bitcoin-charting-library/?feature=technical-analysis-charts)
to request access. Otherwise, attempting to access the GitHub repositories or legacy documentation links will result in the
*404 Not Found* error.

caution

Advanced Charts and Trading Platform should only be downloaded from the official TradingView repositories provided within this documentation. Obtaining the library from third-party services is strictly prohibited and may lead to legal consequences.

The library is not redistributable. Therefore, it is prohibited to use any part of the library in public repositories.

## First steps[​](#first-steps "Direct link to First steps")

You should complete the steps below to get started with Advanced Charts / Trading Platform:

1. [Run Advanced Charts / Trading Platform](/charting-library-docs/v29/tutorials/First-Run-Tutorial) for the first time.
2. Set up [Widget Constructor](/charting-library-docs/v29/core_concepts/Widget-Constructor).
3. [Connect data](/charting-library-docs/v29/connecting_data/) and [implement real-time data streaming](/charting-library-docs/v29/tutorials/implement_datafeed_tutorial/).
4. [Explore](/charting-library-docs/v29/ui_elements/) various parts of the interface and configure them to fit your needs.
5. [Customize](/charting-library-docs/v29/customization/) your chart's appearance.

We recommend that you also check out [Best Practices](/charting-library-docs/v29/getting_started/Best-Practices) and [Frequently Asked Questions](/charting-library-docs/v29/getting_started/Frequently-Asked-Questions) before you start development to avoid common mistakes.

## Integration examples[​](#integration-examples "Direct link to Integration examples")

You can integrate Advanced Charts / Trading Platform with Android WebView, iOS WKWebView, Angular, React, and other frameworks.
Check out the [repository on GitHub](https://github.com/tradingview/charting-library-examples) for some integration examples.

## Online code playgrounds[​](#online-code-playgrounds "Direct link to Online code playgrounds")

You can use the libraries on some code playgrounds/editors such as JSFiddle and Codepen.
The [Online editors](/charting-library-docs/v29/getting_started/Online-Editors) article provides several starter templates,
that can help you quickly create your own examples, reproduce bugs, and experiment with the libraries.

## Hosting the library on a separate origin[​](#hosting-the-library-on-a-separate-origin "Direct link to Hosting the library on a separate origin")

If you want to host an HTML page that contains the chart and the library bundles on different origins (domains), refer to [Cross-origin hosting](/charting-library-docs/v29/getting_started/Hosting-Library-Cross-Origin).
