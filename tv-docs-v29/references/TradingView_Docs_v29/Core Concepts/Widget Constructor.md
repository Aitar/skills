# Widget Constructor

The Widget Constructor is the entry point to the library. It allows you to embed the library within your web page. You can use the Widget Constructor parameters to customize the widget's appearance and behavior. All parameters are listed in the [`ChartingLibraryWidgetOptions`](/charting-library-docs/v29/api/interfaces/Charting_Library.ChartingLibraryWidgetOptions) interface. If you use [Trading Platform](/charting-library-docs/v29/trading_terminal/), you can specify some [additional parameters](#trading-platform-parameters).

The following video tutorial describes Widget Constructor parameters and demonstrates how to use them.
